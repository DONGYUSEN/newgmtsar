# 粗配准多窗口评估报告（LT1B Real SAR (Lutan scene)）

## 结论
- 最高综合得分窗口：`window_size=512, search_range=128`
- 工程推荐窗口：`window_size=384, search_range=96`
- 推荐原则：在接近最优性能条件下优先选择更小窗口，以减少计算开销并保持稳定性。
